5. Room Features 
    #### Messaging
        * Text message
        * Image, video, file upload (with preview)
        * Message edit & delete (with permission)
        * Reply, forward, react (emoji reactions)
        * Threaded replies (optional advanced feature)

    #### User Control (Admin features)
        * Add / remove user
        * Promote / demote user (admin/member)
        * Mute / unmute user
        * Ban user
        * Approve join requests

    #### Privacy & Access
        * Make room **public / private / invite-only**
        * **Access level** per user (read, write, admin)
        * **Join request system** (user requests to join room)
        * **Room password** (optional for protected rooms)

    #### Notifications
        * Typing indicator
        * Read receipts (seen by)
        * Message delivered indicator
        * Mention notification (`@username`)

    #### Real-time Functionality
        * WebSocket / Socket.IO based live updates
        * Message status (sent, delivered, read)
        * Online/offline indicator

    #### Room Meta
        * Room description / topic
        * Room image or banner
        * Room rules or pinned message
        * Last message preview
        * Message count or unread count

    #### Extras (for scalability)
        * **Sub-groups** or “channels” inside a main group
        * **Announcement-only mode** (only admin can send)
        * **Polls / voting system**
        * **File sharing section** (for documents, media)
        * **Search messages by keyword**
        * **Export chat history (JSON/PDF)**


        * **Voice / video chat integration**
        * **Bot support** (auto replies, commands)
        * **Message expiry** (auto delete after X days)
        * **Pinned messages / starred messages**
        * **Room analytics** (active users, message frequency)
